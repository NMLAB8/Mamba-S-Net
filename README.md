## Acknowledgements

This project builds upon ideas from the SCNet architecture for music source separation.

If you find this repository useful, please also consider citing the original SCNet work:

Tong, W., et al.  
**SCNet: Sparse Compression Network for Music Source Separation.**  
arXiv:2401.13276, 2024.  
https://arxiv.org/abs/2401.13276

Our model follows a similar encoder–separation–decoder framework and time–frequency processing pipeline, while replacing the sequence modeling component with a Mamba-based state space model.
